Ans: web page is made of elements, or nodes, such as the html element, the body element, and paragraph elements. The DOM builds a tree-like structure of these nodes in a hierarchy where some nodes are parents, and other nodes are children. Dom makes web page interactive.

4.What is callback funtion ? what is the activity of this fucntion ?
Ans: When a function call another function we can say it callback function.The purpose of a callback function is to allow the first function to invoke a specific behavior defined in the callback function, without having to know the details of how that behavior is implemented.
